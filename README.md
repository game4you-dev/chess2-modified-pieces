# Chess 2 - Modified Cburnett Pieces

This repository contains the modified chess pieces used in the Android game **Chess 2**.

## Original Source
These pieces are based on the famous "Cburnett" SVG chess set, originally created by **Colin Burnett** and hosted on Wikimedia Commons.

## Modifications Made
To fit the visual style of Chess 2, the following modifications were made to the original files:
* Color inversion (swapped black and white colors from the original black pieces).
* Minor contour adjustments (e.g., modified the shape of the Queen).
